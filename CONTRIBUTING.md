## 🇧🇷 Português (Brasil)

### Contribuindo para o KoreAI

Obrigado por seu interesse no KoreAI! Projetos open source prosperam com o apoio da comunidade, e sua ajuda é muito bem-vinda.

#### Como colaborar:

1. **Reportando Bugs:** Use as *Issues* para relatar problemas. Inclua detalhes como a versão utilizada, logs de erro e passos para reproduzir o bug.
2. **Sugestões de Features:** Se tiver uma ideia para o KoreScript ou para a KoreVM, abra uma *Issue* com o rótulo `enhancement` e descreva o caso de uso.
3. **Pull Requests (PRs):**
* Fork o repositório.
* Crie uma branch para sua funcionalidade (`git checkout -b feature/nome-da-feature`).
* Siga o padrão de commits.
* Certifique-se de que os testes estão passando.
* Abra um PR explicando as mudanças.

## 🇺🇸 English

### Contributing to KoreAI

Thank you for your interest in KoreAI! Open source projects thrive on community support, and we highly value your contributions.

#### How to contribute:

1. **Reporting Bugs:** Use the *Issues* tracker to report problems. Please include details such as version, error logs, and steps to reproduce the bug.
2. **Feature Requests:** If you have an idea for KoreScript or the KoreVM, open an *Issue* labeled `enhancement` and describe your use case.
3. **Pull Requests (PRs):**
* Fork the repository.
* Create a feature branch (`git checkout -b feature/feature-name`).
* Follow standard commit conventions.
* Ensure all tests pass.
* Open a PR explaining the changes made.

¡Gracias por tu interés en KoreAI! Los proyectos de código abierto prosperan gracias al apoyo de la comunidad y tu ayuda es muy bienvenida.
